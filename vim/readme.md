### How to prepare a new environment  
1. copy .vimrc to ~/.vimrc  
2. install vundle git clone https://github.com/gmarik/Vundle.vim.git ~/.vim/bundle/Vundle.vim  
3. install black -  pip install black  
4. Markdown preview  
install:   
:PluginInstalli  
:call mkdp#util#install()  

run preview:  
:MarkdownPreview
